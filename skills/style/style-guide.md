# Visual Direction
Editorial minimalist portfolio inspired by Lauren Waller.

# Typography
Primary font:
Inter, Suisse Intl, Neue Haas Grotesk, SF Pro Display, sans-serif

# Colors
Background: #F5F5F3
Secondary Background: #EFEFEA
Primary Text: #111111
Secondary Text: #6B6B6B

# Spacing
Use generous whitespace.
Prefer padding over borders.
Minimum section spacing: 96px desktop / 64px mobile.

# Interaction
Subtle transitions only.
Avoid flashy animations.
